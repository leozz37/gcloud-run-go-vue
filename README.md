# Deploying Containerized Applications on Google Cloud Run

## Golang setup

```bash
go build
```

```bash
./server
```

## Vue setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
